An array is a data structure that stores a collection of elements of the same type in a continuous memory location.

An array is a collection of values stored together, where each value can be accessed using its index.

Example:
numbers = [10, 20, 30, 40]

Index:   0    1    2    3
Value:  10   20   30   40

Index starts from 0
Last index = size - 1

Time Complexity of Array Operations

| Operation            | Complexity |
| -------------------- | ---------- |
| Access by index      | O(1)       |
| Search               | O(n)       |
| Insert at end        | O(1)       |
| Delete/Search remove | O(n)       |

Find maximum → O(n)
Find minimum → O(n)
Reverse array → O(n)
Sum elements → O(n)
