# Time Complexity

## O(1) Constant Time

Example:
arr[0]

## O(n) Linear Time

Example:

for x in arr:
print(x)

## O(n^2) Quadratic Time

Example:

for i in arr:
for j in arr:
print(i,j)

## O(log n)

Example:
Binary Search
